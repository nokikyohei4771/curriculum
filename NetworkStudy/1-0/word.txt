ネットワーク
　
　・LAN　→　限られた範囲内にあるコンピュータや通信機器、情報機器などをケーブルや無線電波などで接続し、相互にデータ通信できるようにしたネットワークのこと

　　・有線LAN　→　室内や建物内の機器を結ぶ構内ネットワークのうち、信号の伝送媒体として通信ケーブルを用いるもの

　　・無線LAN　→　電波による無線通信により複数の機器間でデータの送受信を行う構内ネットワーク


　・WAN　→　地理的に離れた地点間を結ぶ通信ネットワーク

　　・インターネット　→　共通の通信仕様を用いて全世界の膨大な数のコンピュータや通信機器を相互につないだ巨大なコンピュータネットワーク


ネットワーク構成

　・ノード　→　網状構造の構成要素

　・リンク　→　網状の構造物において、構成要素間を結びつける線や経路

　・トポロジ　→　ネットワークを構成する機器同士がどのような規則性に基づいてつながれているかを模式的に表したもの。

　　・バス型トポロジ　→　共用の通信路にすべてのノードを接続する

　　・スター型トポロジ　→　集線装置などを中心に各ノードを接続する

　　・メッシュ型トポロジ　→　各ノードが一つ以上の他のノードと接続する

　・帯域　→　通信などに用いる周波数の範囲

　　・bps　→　通信回線などのデータ伝送速度の単位で、1秒間に何ビットのデータを送れるかを表す

　・輻輳　→　通信回線やネットワークに想定を超える接続要求や伝送要求が行われ、通信通話できなくなる状態


　・コリジョン　→　同じ伝送路を共有する複数の装置が同時に信号を発信して混信してしまう現象

　・ドメイン　→　インターネットなどのTCP/IPネットワーク上で機器やネットワークを識別するドメイン名

　・ポート　→　機器やソフトウェアが外部の別の主体と接続・通信するための末端部分


モデル

　・プロトコル　→　複数の主体が滞りなく信号やデータ、情報を相互に伝送できるよう、あらかじめ決められた約束事や手順の集合のこと
　
　・OSI参照モデル　→　コンピュータネットワークで様々な種類のデータ通信を行うために機器やソフトウェア、通信規約などが持つべき機能や仕様を複数の階層に分割・整理したモデルの一つ

　　・ヘッダ　→　データや文書の本体の先頭に付け加えられるそのデータや文書自体についての情報を記述した部分

　　・カプセル化　→　オブジェクト指向プログラミングにおいて、互いに関連するデータの集合とそれらに対する操作をオブジェクトとして一つの単位にまとめ、外部に対して必要な情報や手続きのみを提供すること

　　・非カプセル化　→　ヘッダを外してデータ部分を取り出すこと

　　・PDU(Protocol Data Unit)　→　データ通信において、あるプロトコルが扱うひとまとまりのデータの送受信単位のこと

　　・MTU(Maximum Transmission Unit)　→　通信機器などが一度に送信できる最大のデータ量

　　・フレーム　→　Webブラウザの表示領域を縦または横に複数の領域に分割してそれぞれに別のページを表示したもの

　　・パケット　→　通信回線やネットワークを流れる情報のうち、データをある長さごとに区切り、送信元や宛先などの制御情報を付加した小さなまとまり

　　・セグメント　→　全体をいくつかに分割したうちの一つ一つの断片や区分のこと

　・TCP/IPモデル　→IPを中心とするTCP/IP群はOSI参照モデルを参照しない、よりシンプルな独自の4階層のモデル


物理層

　・ビット　→　情報量の最小単位で、二つの選択肢から一つを特定する情報の量
　
　・ツイストペアケーブル　→　通信用の金属ケーブルの種類の一つで、細長い金属製の電線を２本1組としてらせん状により合わせ、このペアを何対か集めてゴムなどの被覆材で覆って一本のケーブルとしたもの

　　・UTP(Unshielded Twisted Pair cable)　→　よりあわせた一対の金属線により電気信号や電力を伝える通信用ケーブルの構造の一つで電磁ノイズを遮断するシールド加工を行っていないもの

　　・STP(Shielded Twisted Pair cable) →　よりあわせた一対の金属線により電気信号や電力を伝える通信用ケーブルの構造の一つで、電磁ノイズを遮断するシールド加工を行ったもの

　　・カテゴリー　→　ツイストペアケーブルの標準規格でケーブルの電気特性グレード

　　・コネクタ　→　電気部品やケーブルなどを他の部品に簡単に着脱できるように設けられた部品

　・ストレートケーブル　→　内部に複数の芯線が通っている通信ケーブルのうち、各芯線が両端で同じ位置のピンに結線されているもの

　・クロスケーブル　→　内部に複数の芯線が通っている通信ケーブルのうち、一部の芯線が両端で別の位置のピンに結線されているもの

　・光ファイバーケーブル　→　光ファイバーを用いた通信ケーブル

　・ハブ　→　機器間をケーブルで結んで通信する際に、複数のケーブルを接続して相互に通信できるようにする集線装置、中継装置のこと

　　・半二重通信　→　２者間のいずれの方向へも通信できる双方向通信において、一度に片方しか送信できず、両者が同時に送信することができない方式のこと


・データリンク層

　・イーサネット　

　　・イーサネット　→　主に室内や建物内でコンピュータや電子機器をケーブルでつないで通信する有線LANの標準の一つ

　　・ファストイーサネット　→　イーサネットのうち、100Mbpsでの通信に対応した規格群の総称

　　・ギガビットイーサネット　→　イーサネットのうち、1Gbpsでの通信に対応した規格群の総称

　・イーサネットヘッダ　→　イーサネットフレームのヘッダ部

　・トレーラ　→　データを区切って転送するときに、その区切られたデータ一つ一つに付加される制御情報
　
　・CSMA/CD　→　通信において単一の伝送路を複数の機器が共用する際、中央で監視・制御する機器がなくても伝送路の使用権に調整できる通信方式

　・MACアドレス　→　通信ネットワーク上で各通信主体を一意に識別するために物理的に割り当てられた48ビットの識別番号

　・スイッチ　→　中継・集線装置のうち、データの宛先を見て関連する先にのみ転送を行う機能をもったもの

　　・MACアドレステーブル　→　ネットワークスイッチなどが内部的に管理している情報で、どのポートにどのMACアドレスを持つ機器が接続されているかを記録したもの

　　・フラッディング　→　ネットワーク機器のデータ転送方式の一種やシステムに許容量を超えるデータや処理要求が届き麻痺状態に陥る現象

　　・全二重通信　→　双方向通信において、両者が同時に送信することのできる方式

　・VLAN　→　一つの構内ネットワーク内に物理的な接続形態とは独立に機器の仮想的なグループを設定し、それぞれをあたかも一つのLANであるかのように運用する技術

　　・タグVLAN　→　イーサネットフレームの一部にVLANの識別番号を書き込み、同じVLANのIDを持つポート間に流通させる

　　・ポートベースVLAN　→　一台のLANスイッチのケーブル接続口ごとにどのVLANに所属するか設定する

　　・アクセスポート　→　VLANが設定されたLANスイッチなどで一つのVLANのみに所属するポート

　　・トランクポート　→　VLANが設定されたLANスイッチなどで、複数のVLANに所属し、それぞれのデータを混在させて伝送するポート

　・DTP(DeskTop Publishing)　→　出版物や印刷物の原稿作成や編集、デザイン、レイアウト、組板などの作業をコンピュータで行い、最終的に印刷可能な原稿の作成まで行うこと　

　・デフォルトVLAN　→　VLANを設定したネットワークスイッチでVLANの指定がないポートやフレームの通信に用いられるVLAN

　・ネイティブVLAN　→　IEEE802.1Q方式のVLANでトランクリンク上で唯一タグをつけずに送受信されるVLAN

・ブロードキャストストーム　→　ネットワーク上で一斉配信用の特殊なデータや信号が際限なく転送され続ける現象

・スパニングツリープロトコル　→　冗長構成になっているスイッチドネットワークにおいて、ブロードキャストストームなどのループによる問題を回避するためのプロトコル

・リンクアグリゲーション　→　複数の回線を束ねて一体的に運用し、仮想的に一本の回線とみなすことで通信速度および耐故障性を向上させる技術

・EtherChannel　→　米シスコシステムズ社の通信機器間で複数のイーサネット回線を束ねて仮想的に一本の大容量回線として利用する機能
　

・オートネゴシエーション　→　イーサネット規格のネットワークインターフェースや通信機器が持つ機能の一つで通信相手との間で対応している規格や通信モードの違いを自動的にすり合わせ、最適な設定を行うもの
